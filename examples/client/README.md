## example-client

The simple client for satelit.

```bash
$ client help                                                                                                                                [~/go/src/github.com/lovi-cloud/satelit/examples/client]
satelit client

Usage:
  client [command]

Available Commands:
  create-vm    Create and Start virtual machine
  help         Help about any command
  upload-image Upload image to satelit from qcow2 file

Flags:
      --address string   address of satelit, <host>:9262
  -h, --help             help for client

Use "client [command] --help" for more information about a command.
```
